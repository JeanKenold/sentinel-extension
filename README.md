# 🛡️ Sentinel: Real-Time Privacy Auditor

This project serves as a practical application of theoretical concepts in **Cybersecurity**, **Data Privacy**, and **Software Development**.

Sentinel is a high-performance Chrome Extension designed to prevent data leaks by auditing browser inputs in real-time. Built with a focus on **Application Security** and **Data Integrity**.

## 🚀 Key Features

- **Mathematical Validation:** Implements **Luhn Algorithm** for Credit Cards and **Modulo 11** for CPF (Brazilian Tax ID).
- **High-Fidelity Signature Matching:** Recognizes 15+ API Key patterns (AWS, Stripe, OpenAI, GitHub, etc.).
- **Zero-Knowledge Privacy:** 100% of the analysis is performed locally. No data ever leaves the user's machine.
- **Audit Reports:** Generates professional JSON reports for security compliance and auditing.

## 🛠️ Tech Stack

- **JavaScript (ES6+)** - Content Scripts & Service Workers.
- **Chrome Extensions API (Manifest V3)**.
- **CSS3** - Responsive and adaptive UI with Dark Mode support.

## 🎓 Academic Context

Developed as a flagship project during the **Analysis and Systems Development (TADS)** program at the **Federal University of Paraná (UFPR)**.
